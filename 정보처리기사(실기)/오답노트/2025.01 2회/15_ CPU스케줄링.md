## 15. 준비상태 큐에 각 프로세스의 도착시간과 실행 시간이 다음과 같을 때 RR(Round Robin)로 스케줄링할 때 평균 대기 시간을 쓰시오(단, Time Slice는 4초이다.)

| 프로세스 | 도착시간 | 실행시간 |
| ---- | ---- | ---- |
| A    | 0    | 8    |
| B    | 1    | 4    |
| C    | 2    | 9    |
| D    | 3    | 5    |

<br><br><br><br><br>

sol)
### Round Robin
- 각 프로세스를 시간 할당량 동안만 실행한 후 실행이 만료되지 않으면 다음 프로세스에게 CPU를 넘겨주는 기법

### 대기시간
종료시간 - 실행시간 - 도착시간

**- 프로세스 진행 타임라인**

| 진행시간 | 프로세스 큐 | 실행 프로세스(잔여시간) / 종료 여부 |
| ---- | ------ | --------------------- |
| 4    | ABCD   | A(4)                  |
| 8    | BCDA   | B(0) / 종료             |
| 12   | CDA    | C(5)                  |
| 16   | DAC    | D(1)                  |
| 20   | ACD    | A(0) / 종료             |
| 24   | CD     | C(1)                  |
| 25   | D      | D(0) / 종료             |
| 26   | C      | C(0) / 종료             |

**- 대기시간 계산 표**

| 프로세스 | 도착시간   | 실행시간                  | 종료시간 | 대기시간 |
| ---- | ------ | --------------------- | ---- | ---- |
| A    | 0      | 8                     | 20   | 12   |
| B    | 1      | 4                     | 8    | 3    |
| C    | 2      | 9                     | 26   | 15   |
| D    | 3      | 5                     | 25   | 17   |

- 평균 대기시간
  (12 + 3 + 15 + 17) / 4 = 11.75

**정답: 11.75**
