## 3. 다음 설명에 해당하는 용어를 <보기>에서 찾아 기호(ㄱ~ㅇ)로 쓰시오
> [!NOTE]
> - 데이터 베이스를 구성하는 가장 작은 논리적 단위로, 릴레이션에서 열(Column)에 해당한다.
> - 파일 구조상의 데이터 항목 또는 데이터 필드에 해당된다.
> - 개체의 특성을 기술한다.

```plaintext
ㄱ. Domain  ㄴ. Tuple  ㄷ. Attribute   ㄹ. Integrity
ㅁ. Degree  ㅂ. Relationship  ㅅ. cadinality  ㅇ. Entity
```

<br><br><br><br><br>
sol) 릴레이션의 구성
1. Domain: 하나의 애트리뷰트(속성)이 취할 수 있는 같은 타입의 원자(Atomic) 값들의 집합
2. Tuple: 릴레이션을 구성하는 각 행을 말하며, 파일 구조에서 레코드와 같은 의미
3. Attribute: 데이터베이스를 구성하는 가장 작은 논리적 단위로 속성의 수를 속성의 차수(Degree)라고 한다.
4. Integrity(무결성): 데이터베이스에 저장된 데이터 값과 그 것이 표현하는 현실 세계의 실제 값이 일치하는 정화성(개체 무결성, 참조 무결성, 도메인 무결성)
5. Degree: 속성의 개수
6. Relatoinship(관계)
7. cadinality(카디널리티): 튜플의 수
8. Entity(개체)

**정답: ㄷ**
