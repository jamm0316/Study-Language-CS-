# 1. 다음 설명에ㅓ 해당하는 네트워크 보안관련 용어는?
- 상호 인증과정을 거친 후 접속해있는 서버와 서로 접속되어 클라이언트 사이의 세션 정보를 가로채는 공격 기법이다.
- 접속을 위한 인즈엊ㅇ보 없이도 가로챈 세션을 이용해 공격자가 원래의 클라이언트인 것처럼 위장하여 서버의 자원이나 데이터를 무단으로 사용한다.
- TCp 3-way-handshake 과정에 끼어듦으로써 클라이언트와 서버간의 동기화된 시퀸스 번호를 가로채 서버에 무단으로 접근하느 공격이 대표적이다.

<br><br><br><br><br>
sol) 네트워크 보안
1. 서비스 거부/위장
   - 스푸핑(spoofing): 공격자가 자신을 다른 사람이나 시스템으로 속여서 신뢰를 얻거나 시스템에 접근하는 공격 기법.
   - SYN Flooding: TCP 3-way-handshake를 악용하여 SYN 패킷만 대량으로 전송해 서버 자원을 고갈시키는 서비스 거부 공격.
   - DDos: 다수의 공격자가 동시에 트래픽을 발색이켜 정상적인 서비스 제공을 방해하는 분산 서비스 거부 공격
   - Slowloris: HTTP 연결을 장시간 유지하여 서버의 동시 접속 자원을 소모시키는 서비스 거부 공격
2. 악성코드
   - 웜(Worm): 스스로 복제하고 다른 컴퓨터로 자동으로 퍼져나가느 독립적인 악성 소프트웨어
   - 랜섬웨어(Ransomware): 사용자의 파일을 암호화하여 접근할 수 없도록 만든 후, 이를 해제하기 위해 금전을 요구하는 악성 소프트웨어
   - 백도어(Backdoor): 시스템이나 소프트웨어에 몰래 설치된 비정상적인 접근 경로
   - 트로이 목나(Torijan Horse): 정상 프로그램으로 위장하여 사용자가 실행하기 위해 설치를 유도한 뒤 악성행위를 수행하는 악성코드
3. 네트워크 침해
   - 세션 하이재킹: 인가된 세션 자체를 훔치는 것이 목적
   - 스위치 재밍: 위조 MAC 주소로 스위치를 혼란시켜 트래픽 도청
   - 스미싱/피싱: 정보 탈취 목적의 사회공학적 네트워크 공격
   - Evil Twinn Attack: 가짜 WiFi로 로그인 정보 탈취
