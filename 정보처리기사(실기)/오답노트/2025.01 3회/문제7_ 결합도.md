  ## 7. 다음은 결합도(Coupling)에 대한 설명이다. 괄호(1~3)에 들어갈 알맞은 결합도를 <보기>에서 찾아 기호(ㄱ~ㅂ)로 쓰시오.
  - (1): 한 모듈이 다른 모듈의 내부 기능 및 그 내부 자료를 직접 참조하거나 수정할 떄의 결합도이다.
  - (2): 모듈 간의 인터페이스로 배열이나 레코드 등의 자료 구조가 전달될 때의 결합도이다.
  - (3): 공유되는 공통 데이터 영역을 며러 모듈이 사용할 떄의 결합도록, 파라미터가 아닌 모듈 밖에 선언된 전역 변수를 사용하여 전역 변수를 갱신하는 방식의 상호작용하는 때의 결합도이다.

<보기>
```plaintext
ㄱ. 외부 결합도  ㄴ. 스탬프 결합도  ㄷ. 제어 결합도
ㄹ. 자료 결합도  ㅁ. 내용 결합도   ㅂ. 공통 결합도
```

<br><br><br><br><br>
sol)
| 결합도     | 키워드           | 설명                                  |
| ------- | ------------- | ----------------------------------- |
| 내용 결합도  | 내부 직접 참조/수정   | 다른 모듈의 내부 기능·자료를 직접 접근하거나 수정        |
| 공통 결합도  | 전역 변수(공통 영역)  | 여러 모듈이 공유 데이터 영역(전역변수)을 함께 사용       |
| 외부 결합도  | 외부 선언 변수      | 한 모듈이 선언한 변수를 외부 다른 모듈이 참조          |
| 제어 결합도  | 제어 신호(Flag)   | Flag, Switch 등 제어 요소로 상대 모듈의 흐름을 제어 |
| 스탬프 결합도 | 자료 구조(배열/레코드) | 배열, 레코드 등 구조체를 통째로 전달               |
| 자료 결합도  | 자료요소(값)       | 모듈 간 단순 데이터(값)만 주고 받음               |

**정답
1: ㅁ
2: ㄴ
3: ㅂ**
