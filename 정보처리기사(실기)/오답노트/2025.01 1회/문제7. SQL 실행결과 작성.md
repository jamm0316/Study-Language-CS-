## 7. <A>테이블과 <B>테이블을 참고하여 <SQL 문>의 실행 결과를 쓰시오.
| NAME  | RULE |
| ----- | ---- |
| Smith | S%   |
| Allen | %T%  |
| Scoot |      |

<SQL 문>
```sql
SELECT COUNT(*) CNT FROM A CORSS JOIN B
WHERE A.NAME LIKE B.RULE
```
<br>
<br>
<br>
<br>
<br>

sol)
1. `FROM A CROSS JOIN B`: <A>와 <B>를 고차 조인한 결과를 대상으로 검색한다.

| NAME  | RULE | Note |
| ----- | ---- | ---- |
| Smith | S%   | 1    |
| Smith | %T%  | 2    |
| Allen | S%   |      |
| Allen | %T%  |      |
| Scoot | S%   | 3    |
| Scoot | %T%  | 4    |

이 중, `WHERE A.NAME LIKE B.RULE` 에 해당 하는 튜플은 1, 2, 3, 4 튜플이다.

**정답: 4**
