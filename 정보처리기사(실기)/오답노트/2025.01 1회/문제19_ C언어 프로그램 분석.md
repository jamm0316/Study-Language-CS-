## 19. 다음 C언어로 구현된 프로그램을 분석하여 그 실행결과를 쓰시오.(단, 출력문의 출력서식을 준수하시오.)

```C
#include <stdio.h>
struct Node {
    struct Node* next;
    unsigned int x;
}

int main() {
    struct Node t1 = {0, 5u};
    struct Node t2 = {0, 7u};
    struct Node t3 = {0, 11u};
    t3.next = &t2;
    t2.next = &t1;
    struct Ndoe* curr = &t3;
    int sum = 0;
    while (curr) {
        sum = sum * 3 + curr -> x;
        curr = curr -> next;
    }

    sum = (sum^42u) + 100u;
    printf("%u/n", sum);
}
```

<br><br><br><br><br>
sol) <br>
1. Node의 주소값을 가리키는 next와, 음수를 제외한 정수를 저장하는 unsigned int 타입의 x를 필드로 갖는 Node struct를 선언한다.
2. 각각 주소값과 정수값으로 초기화한 t1, t2, t3을 선언한다.
3. t3의 next는 t2로 저장한다.
4. t2의 next는 t1으로 저자한다.
5. Node 포인터 타입의 curr에 t3의 주소값을 저장한다.
6. curr 주소값이 0(false)가 아니면 반복문을 계속한다.
  - 1번째 순회(sum = 0, curr -> x = 11)<br>
    sum = 0 * 3 + 11 = 11;<br>
    curr = t3.next = t2<br>
  - 2번째 순회(sum = 11, curr -> x = 7)<br>
    sum = 11 * 3 + 7 = 40;<br>
    curr = t2.next = t1<br>
  - 3번째 순회(sum = 40, curr -> x = 5)<br>
    sum = 40 * 3 + 5 = 125;<br>
    curr = t1.next = 0<br>
7. sum = 125u^42u + 100u; => 125와 42의 XOR 연관관계 (서로 같으면 0, 다르면 1)<br>
  
  | 10진수 | 2^7 | 2^6 | 2^5 | 2^4 | 2^3 | 2^2 | 2^1 | 0   | 비고 |
  | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
  | 42 | 0   | 0   | 1   | 0   | 1   | 0   | 1   | 0   | - |
  | 125 | 0   | 1   | 1   | 1   | 1   | 1   | 0   | 1   | - |
  | 87 | 0   | 1   | 0   | 1   | 0   | 1   | 1   | 1   | XOR |
  <br>
8. 87u + 100u = 187u
<br>
**정답: 187**
