## 12. 다음 <R>과 <S> 테이블에 대해 'R ÷ S'를 실행 했을 때 출력되는 결과를 테이블로 작성하시오.

| A   | B   |
| --- | --- |
| a1  | b1  |
| a2  | b2  |
| a1  | b3  |

| B   |
| --- |
| b1  |
| b3  |

<br>
<br>
<br>
<br>
<br>

sol) 순수 관계 연산자
 <br>
- SELECT: σ(시그마) = 수평연산
  : 릴레이션에 존재하는 튜플 중 선택 조건을 만족하는 튜플의 부분집합을 구하여 새로운 릴레이션을 만드는 것
  <br>
- PROJECT: π(파이) = 수직연산
  : 주어진 렐레이션에서 속성 리스트(Attribute List)에 제시된 속성값만을 구출하여 새로운 릴레이션을 만든느 연산
  <br>
- JOIN: ⋈ = Cartesion Product(교차곱) -> SELECT
  : 공통 속성을 중심으로 두개의 릴레이션을 하나로 합쳐서 새로운 릴레이션을 만드는 연산
  <br>
- DIVISION: ÷
  : X ⊃ Y인 두개의 릴레이션 R(X)와 S(Y)가 있을 때, R의 속성이 S의 속성값을 모두 가진 튜플에서 S가 가진 속성을 제외한 속성만 구하는 연산
<br>

1. R의 속성에서 S의 속성값을 모두 가진 튜플 <br>
| A   | B   |
| --- | --- |
| a1  | b1  |
| a1  | b3  |

<br>

2. S가 가진 속성을 제외한 속성
| A   |
| --- |
| a1  |
| a1  |

<br>

따라서,
**정답:**
| A   |
| --- |
| a1  |
