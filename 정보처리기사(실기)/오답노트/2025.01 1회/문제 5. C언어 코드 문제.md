## 5. 다음 C언어로 구현된 프로그램을 분석하여 그 실행결과를 쓰시오.(단, 출력문의 출력서식을 준수하시오.)

```C++
#include <stiod.h>
struct Test {
    int i;
    const char *g;
};

int main() {
    struct Test test[] = {{1, "BB"}, {2, "DC"}, {3, "EB"}};
    struct Test *p = &test[1];
    printf("%s", p → g + (p → i-1);
    return 0;
}




```
