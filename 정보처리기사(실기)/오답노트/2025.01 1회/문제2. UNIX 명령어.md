문제2. 다음 설명에 해당하는 UNIX 명령어를 <보기>에서 찾아 쓰시오.
- (1): 현재 작업중인 디렉터리 경로를 화면에 표시한다.
- (2): 현재 디렉터리 파일목록을 표시한다.
- (3): 디렉터리 위치를 변경한다.
- (4): 파일을 복사한다.

> [!NOTE]
> <보기> <br>
> `ls    fork   ps       cp` <br>
> `cat   pwd    chmod    cd`

<br>

sol) UNIX/LINUX 기본 명령어
- cat: 파일 내용 표시 (=Concatenate, 사슬 같이 잇다)
- cd: 디렉터리 위치 변경 (=Change Directory)
- fork: 새로운 프로세스 생성
- ls: 현재 디렉터리 파일 목록 표시 (=list)
- ps: 현재 실행 중인 프로세스 표시 (=process status)
- cp: 파일을 복사함 (=copy)
- pwd: 현재 작업중인 디렉터리 경로 표시 (=print working directory)
- chmod: 파일 보호모드를 설정하여 파일 사용허가를 지정 (=change mode)


정답
1. pwd
2. ls
3. cd
4. cp
